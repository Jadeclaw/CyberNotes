wmic permet de retourner des informations système

Exemples :

wmic process list full # affiche la liste complète de tout les processus du système

wmic process get name,parentprocessid,processID # affiche tout les processus et leurs parents

wmic process where "processID=736" get commandline # affiche le chemin du processus N°736 (là ou il est stocké)

wmic startup get caption,command # affiche les tâches lancées au démarrage du système
