permet de rechercher des fichiers dans notre disque

Exemples :

forfiles /D -10 /s /m *.exe /c "cmd /c echo @path" >> exefiles.txt # permet d'aficher les .exe dans

forfiles /P C:\ /s /m *.txt # affiche tout les .txt dans C:

forfiles /d -10 /s /m *.exe /c "cmd /c echo @ext @fname @fdate" # affiche un tri détaillé des .exe dans C:
