affiche les processus en cours d'exécution sur l'ordinateur

Exemples :

tasklist >> task.txt # affiche le rapport classique des process exécutés .exe


tasklist /m >> taskmod.txt # affiche en plus, les modules rattachés à chaque processus


tasklist /svc >> taskservice.txt # affiche en plus, les services rattachés à chaque processus


.dll = librairie partagée : certaines ne sont pas signées et peuvent être d'une source frauduleuse.
par exemple, un programme qui lis dans les .dll d'un autre programme à des fins malveillantes etc...
ceci est valable aussi pour les services qui peuvent aussi être malveillants.
